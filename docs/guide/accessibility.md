# Accessibility Guide

EveryUI prioritizes accessibility in every component.

- Semantic HTML tags
- Keyboard-friendly interactions
- Visible focus rings with `:focus-visible`
- Form labels and hint/error wiring via `FormField`
- `Dialog` includes `role="dialog"` and `aria-modal="true"`
- Toast messages use `aria-live="polite"`

Always test your own content with keyboard and screen readers.
