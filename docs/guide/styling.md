# Styling Guide

EveryUI ships with strong defaults so you can skip most CSS.

## Theme tokens

- Colors
- Spacing
- Radius
- Shadows
- Typography
- Motion

You can override CSS variables at `:root` to re-theme your app.

```css
:root {
  --everyui-color-primary: #5b21b6;
  --everyui-radius-md: 0.85rem;
}
```
